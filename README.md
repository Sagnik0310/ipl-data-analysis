# 🏏 IPL Data Analysis (2008–2025)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Indian Premier League (IPL) data from 2008–2025 using Python, Pandas, NumPy, and Matplotlib.

The objective is to analyze team performance, player statistics, venue trends, and season-wise patterns through data visualization and statistical analysis.

---

## 📊 Dataset

The project uses:

* `matches.csv`
* `deliveries.csv`

### Dataset Statistics

* Total Matches: **1158**
* Total Deliveries: **134,190**

* Certain IPL franchises have consistently dominated the league.
* Top batters have maintained exceptional run-scoring records.
* Leading bowlers have taken a significant number of wickets across seasons.
* Some venues host considerably more IPL matches than others.
